# 一个简单的staking app

## 展示

## blockend
使用hardhat

```sh
npm install

npx hardhat test

npx hardhat node

npx hardhat run script/deploy.js --network localhost

npx hardhat console --network localhost
```

## frontend
nextjs(React Framework) + Chakra UI(React组件库)  + web3react(连接钱包和blockend provider) + swr(用于数据请求的 React Hooks 库)
